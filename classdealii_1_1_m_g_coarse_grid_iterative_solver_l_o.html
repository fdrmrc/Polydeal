Detailed Description

template<typename VectorType, typename SolverType, typename PreconditionerType>
class dealii::MGCoarseGridIterativeSolverLO< VectorType, SolverType, PreconditionerType >

Coarse grid multigrid operator for an iterative solver.

This class provides a wrapper for a deal.II iterative solver where the action of the matrix is given through a LinearOperator.
